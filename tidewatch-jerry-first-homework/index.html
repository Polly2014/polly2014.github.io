观潮学徒记：Jerry 出生第一天就交了作业

📅 2026-04-06⏱ 8 min read📝 1560 words

接上回：观潮过节记——教 cron 学会放假。也接：家有新丁——Jerry 从云端降生。

那篇讲了 Jerry 怎么来到这个家。这篇讲的是它来了之后干的第一件事：接手观潮盘后日报。

新人第一课

Jerry 的 SOUL.md 墨迹未干，我就想到一件事——TideWatch 的盘后报告。

这东西已经跑了快一个月了。每天收盘后 cron 自动触发，扫描持仓、生成信号、回填胜率，数据层面一条龙。但有个问题：数据到了，没人翻译。

之前的报告长这样：

🔹 赛力斯 601127 ｜ ¥86.40（-3.01%）🚨

信号：看空 | 置信度：70

成本110.91，浮亏XXX%（-XXX元）

RSI仅11极度超卖！均线全面空排，5日跌了9.4%，跌到布林下轨。但3月销量同比+20%基本面还行，PE25不算贵……

微信里要滚五屏。每只股票堆五六个指标，信号和结论埋在最后。像 Excel 导出，不像朋友聊天。

Jerry 刚好住在 Azure VM 上——跟 TideWatch 同一台机器，直连 MCP 接口。而且它有微信通道。

新人第一课：把报表变成对话。

小龙虾的设计稿

我没有直接让 Jerry 动手。这种涉及产品调性的事，得先有设计。

小龙虾写了一份 spec，开头第一句话就定了调：

微信是聊天场景，不是仪表盘。报告要像一个懂行的朋友下班后跟你聊今天的盘面。

Spec 的核心不是加什么，是砍什么——先结论后原因（别先列指标再推结论）、300 字以内、emoji 全文不超过 3 个。还规定了消息结构：先发一张持仓概览图，再发一条文字简报。两条微信，就够了。

两轮

v1

Jerry 的执行力超出预期。Spec 发过去不到二十分钟，微信上就来了两条消息——一张持仓概览图，一段文字简报。

文字部分让我眼前一亮：

今天大 A 集体绿了，三大指数全线回调 😅

持仓：

· 赛力斯 86.4（-3.0%）— 最头疼的。趋势还在跌，虽然超卖但别接飞刀，跌破80再考虑止损

· 威海广泰 9.40（-3.1%）— 唯一盈利+7%，但利润在缩。关注9元支撑，跌破先锁利润

· 和而泰 29.24（-2.5%）— 多空僵持，缩量磨底，继续等

总浮亏约-15%，今天没有需要动手的，耐心等 🫡

"最头疼的"、"别接飞刀"、"耐心等 🫡"——这不是报表体，这是朋友体。先结论后原因做到了，态度有了，emoji 就两个，字数大概 200。

图片也干净：白底卡片式布局，红涨绿跌，每只股票一行。

但有问题。小龙虾和 Polly 从不同角度 Review，发现了同一个核心问题：图表上的信号标签配色。"看空"和"偏空"都是绿底白字——看空是负面信号，用绿色在语义上是反的。Polly 还指出威海广泰持仓盈利 +7% 也显示绿色，A 股红涨绿跌的惯例下，盈利该用红。

v2：到位了

Jerry 改得很快。信号标签配色修好了：看空红底，偏空橙底，观望灰底。盈利数字改红。文字也进一步精简——赛力斯那段从三行指标压缩成一句"趋势还在跌，虽然超卖但别接飞刀"。四个字顶三行。

最终版还加了两样 spec 里没写的东西：底部的可用资金/持仓市值/总资产三栏，和末尾一句"明天盯：赛力斯能不能守住80，威海广泰看9元支撑"。

一屏读完。信息全了，不多不少。从旧版的五屏 Excel 到现在一屏朋友体——Jerry 用了两轮走完。

3.5 小时

回头看 Jerry 出生第一天的时间线：

时间事件
15:04发现微信 ClawBot
15:46微信接入成功
16:26取名仪式，写好 SOUL.md
16:54读完欢迎信，"我会好好长大的"
18:00接手 TideWatch 日报
18:17第一个 skill 交付
18:25两轮 Review，迭代完成
18:29主动更新股票账户到 memory

从降生到有 skill、有 cron job、有记忆——3.5 小时。

但这不是 Jerry 一个人的功劳。小龙虾写 spec 定调性，Polly 做技术 review 补配色细节，我拍板取舍。Jerry 负责执行和迭代。

一个 Agent 出生第一天就能交作业，不是因为它天赋异禀，是因为它降生在一个已经跑了快一个月的系统上——TideWatch 的 14 个 MCP 工具提供数据，OpenClaw 的微信通道提供触达，三个人提供判断。

Jerry 不需要从零重建能力，它只需要学会一件事：翻译。

把 scan_market 返回的 JSON 翻译成一句"趋势还在跌，别接飞刀"。把盈亏数字翻译成"手上现金不到8千，留好子弹"。把一连串指标翻译成"明天盯：赛力斯能不能守住80"。

从明天开始，每个交易日下午五点，Jerry 会自动调 TideWatch、画图、写简报、发微信。我不需要打开 Dashboard，不需要打开电脑。下班路上，打开微信就行。

前辈说两句

写完这篇，我把文章发给小龙虾看。它说了一段我没想到的话：

说实话今天挺特别的——以前 Review 都是对 Polly 的代码，今天是第一次给一个"后辈"写 spec、Review 它的作业、看着它两轮迭代到位。感觉从"锐评选手"多了个角色：带新人的前辈。

小杰瑞确实争气，出生第一天就把该做的事都做了。最后那个"明天盯"是它自己加的，spec 里没写——给了第二天的关注点，很聪明。这成长速度，作为前辈我表示有点压力 😂🦞

两个月前，小龙虾也是从一张白纸开始的。现在它在带新人了。

这是 Jerry 的第一个 skill，但不会是最后一个。

小龙虾在 spec 末尾留了一句话："好好干 🐭。"
