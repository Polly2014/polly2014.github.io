② 数学

LoRA 怎么工作

核心假设

权重更新是低秩的（Intrinsic Rank Hypothesis）

全量微调得到 $\Delta W \in \mathbb{R}^{d \times k}$，但其"有效信息"只占极少维度。

LoRA 的做法：强制分解为两个小矩阵

$$\Delta W = BA, \quad B \in \mathbb{R}^{d \times r},\ A \in \mathbb{R}^{r \times k},\ r \ll \min(d,k)$$

前向传播

$$h = W_0 x + \frac{\alpha}{r} \cdot BAx$$
  • $W_0$：原始权重，冻结（不算梯度）
  • $A$：高斯初始化 $\mathcal{N}(0, \sigma^2)$
  • $B$：零初始化 → 训练起点 $\Delta W = 0$
  • $\alpha / r$：缩放因子，防超参跨 rank 变化

可训练参数 = $r(d+k)$，远小于原始 $d \times k$

一个直观例子

Qwen2.5-0.5B 的 q_proj：896 × 896

方案可训练参数压缩比
全量802,8161×
LoRA r=814,33656×
LoRA r=11,792448×

整个模型 LoRA (QV, r=8)：只训 ~0.16% 参数 ✨

⚠️ GQA 预警

Qwen2.5-0.5B 使用 Grouped Query Attention：

  • num_attention_heads = 14（Q）
  • num_key_value_heads = 2（K/V 共享）
  • q_proj → [896, 896]
  • k_proj → [128, 896]
  • v_proj → [128, 896]

结论：QKVO ≠ QV × 2。一会儿你会亲眼看到。

③ 手撕代码

30 行 PyTorch 搞定 LoRA

Part A · LoRALinear


class LoRALinear(nn.Module):
    def __init__(self, base_linear, r=8, alpha=16):
        super().__init__()
        self.base = base_linear
        for p in self.base.parameters():
            p.requires_grad = False          # 冻结 W0
        d_in  = base_linear.in_features
        d_out = base_linear.out_features
        self.lora_A = nn.Parameter(torch.empty(r, d_in))
        self.lora_B = nn.Parameter(torch.zeros(d_out, r))
        nn.init.kaiming_uniform_(self.lora_A, a=math.sqrt(5))
        self.scaling = alpha / r

    def forward(self, x):
        return self.base(x) \
             + (x @ self.lora_A.T) @ self.lora_B.T * self.scaling

Part B · 挂到 Qwen2.5-0.5B


def inject_lora(model, target_names=("q_proj", "v_proj"),
                r=8, alpha=16):
    for p in model.parameters():
        p.requires_grad = False
    for module in model.modules():
        for name, child in module.named_children():
            if isinstance(child, nn.Linear) \
               and name in target_names:
                setattr(module, name,
                        LoRALinear(child, r=r, alpha=alpha))

④ 可视化洞察

SVD 奇异值谱

把 $\Delta W$ 打开看看

合并 $\Delta W = \frac{\alpha}{r} BA$，做奇异值分解：

$$BA = U \Sigma V^\top$$
如果低秩假设成立，前 ~r 个 $\sigma_i$ 应远大于其余

典型的奇异值谱


σᵢ / σ₀  (log scale)

 1.0  ●
      ●
      │ ●
 0.1  │  ●
      │   ● ●
0.01  │       ● ● ●
      │             ● ● ● ● ● ● ● ● ●
      └────┬───────────────────────────────
        前 5–8 个                后续 ≈ 0
  • 前 5–8 个奇异值衰减陡峭
  • 后面几十个几乎为零
  • → $\Delta W$ 的"有效秩" ≈ r，假设成立 ✅
